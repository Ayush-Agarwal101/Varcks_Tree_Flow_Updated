# Project Specification

## User Requirement
build a backend for online bakery shop

## Selected Technology Stack

### 1. Web Development

### 2. Backend
**Why chosen:** backend
**Purpose:** online bakery shop

### 3. REST
**Why chosen:** simple and widely supported
**Purpose:** API design for the bakery shop

### 4. Python
**Why chosen:** Python has extensive libraries for handling APIs and can be easily integrated with web frameworks like Flask or Django.
**Purpose:** To build a RESTful API backend efficiently.

### 5. FastAPI
**Why chosen:** FastAPI is modern and has a fast development cycle.
**Purpose:** To build a backend with high performance and ease of use.

### 6. Uvicorn
**Why chosen:** lightweight ASGI server
**Purpose:** fast startup and shutdown

## Technology Stack Summary
Web Development → Backend → REST → Python → FastAPI → Uvicorn
